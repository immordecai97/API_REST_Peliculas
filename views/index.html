API de pelis

RUTAS:

Nota: todas las rutas tienen por defecto despues del "http://localhost:3000" llevan el siguiente endpoint: "/api/v1"

por lo tanto quedaría de la siguiente manera: "http://localhost:3000/api/v1" más el endpoint

Rutas GET

  • http://localhost:3000/api/v1/peliculas Nota: Si no estas "loggeado/a" te va a responder con un "Acceso denegado"
  • http://localhost:3000/api/v1/peliculas/:id
  • http://localhost:3000/api/v1/usuarios

Rutas de POST

  • Crear una pelicula
    • http://localhost:3000/api/v1/peliculas
    • Body:

      {
      "title": "Una linda pelicula nueva",
      "year": 2024,
      "director": "La profe Camila",
      "duration": 142,
      "poster": "https://i.ebayimg.com/images/g/4goAAOSwMyBe7hnQ/s-l1200.webp",
      "genre": ["Drama"],
      "rate": 10
      }

  • Crear un usuario
    • http://localhost:3000/api/v1/usuarios
    • Body:

      {
      "nombre": "Camila",
      "password": "unaContrasenaDificil",
      "email": "camila@gmail.com",
      "estado": "true",
      "imagen": "nezuko.jpg"
      }

  • Iniciar sesión
    • http://localhost:3000/api/v1/login Nota: Al iniciar sesión te retornará el token para luego pasarlo por header como "auth"
    • Body:

      {
      "email": "carlos@gmail.com"
      "password": "1234",
      }

Rutas de PATCH

  • Modificar una pelicula
    • http://localhost:3000/api/v1/peliculas/:id
    • Body:

      {
      "title": "Una linda pelicula MODIFICADA",
      "year": 2020,
      "director": "Akira Toriyama",
      }

Rutas de PUT

  • Modificar usuario
    • http://localhost:3000/api/v1/usuarios/:email
    • Body:

      { "email": "carlitos@gmail.com" "nombre": "Nuevo nombre" "password": "123456" }

Rutas de DELETE

  • Eliminar pelicula
    • http://localhost:3000/api/v1/peliculas/:id
